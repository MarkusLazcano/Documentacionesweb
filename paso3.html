Documentación de la práctica TESEBASA

Importar Base de Datos

Paso 1 :

Activamos los servicios Apache y MySQL

Paso 2 :

Entramos al SGBD Workbench

Paso 3 :

Entramos al Local Instance , luego nos vamos a Server y luego seleccionamos DataExport,seleccionamos la Bd a exportar y este nos solicitara nuestra contraseña de root

Paso 4 :

En la pagina principal de Workbench le damos click al simbolo ( + ) y ponemos la ip del servidor

Esperamos a que se termine de exportar la base de datos

Volvemos a la pagina principal de Workbench y entramos a la conexion del servidor, este nos solicitara la contraseña de acceso

Verificar la exportacion :

Creamos una BD con el nombre exacto de la que exportamos anteriormente,nos dirigimos a Server y luego a Data Import , seleccionaremos en donde se exporto la BD , despues de esto seleccionamos abajo cual es la base de datos y le daremos al boton import.

Paso 5 :

Verificamos desde Putty con el comando "show database_ " , este nos debe mostrar la Base de Datos Correctamente importada
